# Calculadora de Descontos - Python 🛒

Este projeto é um script simples em Python desenvolvido para praticar lógica de programação, o uso de funções com parâmetros e tratamento de exceções (`try/except`). O sistema calcula o valor final de uma venda com base no perfil do cliente.

## Funcionalidades

O programa solicita o valor da compra e o tipo de cliente, aplicando os seguintes descontos:
- **Funcionário:** 30% de desconto (multiplicador 0.70).
- **Cliente VIP:** 20% de desconto (multiplicador 0.80).
- **Cliente Comum:** Valor integral (com mensagem de incentivo para aderir ao cartão da loja).

## Tecnologias e Conceitos Utilizados

- **Linguagem:** Python 3
- **Paradigma:** Programação Estruturada
- **Conceitos Aplicados:**
  - Funções com múltiplos parâmetros.
  - Estruturas condicionais (`if`, `elif`, `else`).
  - Tratamento de erros para garantir que apenas números sejam digitados.
