# Sistema de Cálculo de Descontos 🛒

Este é um script simples em Python desenvolvido para praticar lógica de programação, estruturas condicionais e tratamento de exceções. O programa calcula o valor final de uma venda baseando-se no tipo de cliente.

## 🚀 Funcionalidades

- Cálculo de desconto automático:
  - **Funcionário:** 30% de desconto.
  - **Cliente VIP:** 20% de desconto.
  - **Cliente Comum:** Sem desconto (com convite para fazer o cartão da loja).
- Tratamento de erros para entradas inválidas (letras em vez de números).

## 🛠️ Tecnologias

- **Python 3**
- Estruturas de repetição/decisão e funções.
